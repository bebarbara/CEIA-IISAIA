Conceptos del Frontend

Explorá una página que se enseña a sí misma. Cada sección demuestra un concepto y al mismo tiempo está construida con ese concepto.

Estructura Estilo Estado DOM Single-file

1. Estructura — HTML semántico

Las etiquetas son sustantivos. La página entiende de qué trata por los nombres.

2. Estilo — Layout y variables CSS

El CSS responde a "cómo se ven". Layout y variables son las dos perillas que vale la pena nombrar.

3. Estado y comportamiento

Estado es el dato detrás. DOM es lo que ves. Eventos son el puente.

4. El DOM — el árbol vivo

El HTML es texto. El DOM es la representación viva en la memoria del navegador.

5. Single-file: todo en un archivo

Esta página entera vive en un solo .html. Vela.
