# 🎂 Landing Parodia — Torta de cumpleaños

## Descripción

Esta página es una landing page parodia que simula el cálculo “perfecto” de cómo cortar una torta de cumpleaños según la cantidad de invitados.

El usuario ingresa una cantidad de personas (del 1 al 30) y, al hacer click en el botón **Calcular**, la aplicación genera una cantidad aleatoria de porciones sobre la torta.

Cada porción aparece visualmente marcada en la torta con su número correspondiente, creando un efecto divertido y exageradamente “científico”. 😊

---

## Tecnologías utilizadas

* HTML5
* CSS3
* Vanilla JavaScript

Sin frameworks ni dependencias externas.

## Funcionalidades implementadas

✅ Input para cantidad de invitados (1 al 30)

✅ Botón “Calcular”

✅ Generación aleatoria de porciones

✅ Renderizado visual de las divisiones de la torta

✅ Numeración de cada porción

✅ Responsive para mobile

✅ Animaciones suaves y estética interactiva

---

## Qué funcionó bien

* El render dinámico de las porciones
* La distribución circular de los números
* El estilo visual romántico y divertido
* La interacción simple y rápida
* La compatibilidad mobile

---

## Qué no funcionó tan bien

* Cuando aparecen muchas porciones (más de 20), algunos números pueden quedar muy juntos
* La cantidad de porciones es completamente aleatoria y no tiene lógica matemática real 😄
* El sistema no calcula tamaños reales de porción
